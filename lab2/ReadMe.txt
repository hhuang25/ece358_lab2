To compile the program, please run the following command in the console:
"make"

Then run the scripts:
./run_ABP.sh
./run_ABP_NAK.sh
./run_GBN.sh

You will get a message that the script has begun.
You will get another message when the script finishes.

They scripts will generate:
ABP.csv
ABP_NAK.csv
GBN.csv
